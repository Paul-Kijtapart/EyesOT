# EyesOT
A Crime tracker Server helps notify police of up-coming dangerous events.

## Set up environment
1. Javascripts
```
npm install
npm run webpack
```

2. CSS
```
gem update --system
gem install compass susy breakpoint compass-normalize
compass watch
```

## Debugging commands:
```
npm run webpack
compass watch
```

## Where to see the final result?
Go to localhost:3000 to see.
